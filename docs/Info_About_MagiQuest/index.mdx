# Info About MagiQuest

**MagiQuest** is a live-action role playing game where players explore a fantasy world as magic users, known as [Magi](docs\Info_About_MagiQuest\Magi.md). The game is an attraction played at specific [real world locations](docs\Realm_Locations\index.mdx); where Magi use an electronic magic [wand](docs\Info_About_MagiQuest\Game_Mechanics\Wands.md) with IR technology to interact with objects.
